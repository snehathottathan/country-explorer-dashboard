## Features

- **Fetch countries**: Fetches country data from an external API.
- **Search**: Filter countries by name.
- **Region filter**: Filter by continent (Africa, Americas, Asia, Europe, Oceania).
- **Population filter**: Filter countries by population size (Small, Medium, Large).
- **Pagination**: Navigate through countries in pages of 12 items each.
- **Favorites**: Mark/unmark countries as favorites and manage them.
